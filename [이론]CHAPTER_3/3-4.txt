3-4 Box Model

[박스의 4요소]
1. Contents
2. Border - 경계
3. Padding - 박스의 보더에서 안쪽으로 있는 간격
4. Margin - 박스의 보더에서 바깥쪽으로 있는 간격

[단축키]
- padding/margin의 상하좌우 값이 같을 때
ex)
padding:20px

- padding/margin의 상=하, 좌=우일 때
ex)
padding:20px(상하) 10px(좌우)

- padding/margin을 상-우-하-좌 방향으로 값을 주고 싶을 때
padding:20px(상) 10px(우) 15px(하) 5px(좌)
=> 시계방향

- border 작성
폭, 스타일, 색상 순서대로 작성
ex)
border:5px dashed red;
